Email Access
 

Um Ihre Emails mit einem Browser zu lesen verwenden Sie bitte den Web-Zugang den das kiz zur Verfügung stellt.

Die Management -Funktionen um z.B. das Passwort zu ändern oder um Nachrichten weiterzuleiten stehen ebenfalls via Web-Anwendung zur Verfügung.

Zur Administration und zum Zugriff auf die Mailing-Listen, die über diesen Server laufen, gibt es auch eine Web-Anwendung.

To access your mailbox with a browser please use the Web frontend provided by the kiz.

Management functions like changing the password or setting up mail forwarding are also available from the Web.

There is also a Web frontend for accessing and administrating the mailing lists, hosted on this server.
